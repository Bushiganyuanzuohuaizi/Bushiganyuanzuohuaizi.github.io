# Contributing

欢迎来到 `牢邓的翘课日记`。

这个仓库主要用来维护站点内容、导航结构和页面体验，也会承载和其他站点的联动页面，目前已经预留了 `EndeavourCHN` 的接入位置。

## 你可以贡献什么

- 修正文案、错字、失效链接
- 补充新的学习笔记或整理旧笔记
- 调整 `mkdocs.yml` 导航结构
- 优化首页、专题页和阅读体验
- 增加新的外站集成页或友情链接页

## 基本约定

1. 新增文章时，把 Markdown 文件放到 `docs/` 下对应目录。
2. 新增页面通常需要同步更新 `mkdocs.yml` 里的 `nav`。
3. 文件开头尽量加上最简 front matter，例如：

```md
---
title: 页面标题
---
```

4. 图片、附件尽量和所属文章放在相近目录，避免散落。
5. 如果内容引用了别人的文章、图表或项目，请保留来源信息。

## 推荐流程

1. 先确认你要改的是内容、结构还是样式。
2. 小改动直接修改对应文件。
3. 新增栏目时，优先补一个 `index.md` 作为栏目首页。
4. 本地预览站点，确认导航、链接和排版正常。
5. 再提交变更。

## 关于联动页

如果你要把别人的网站接入本站，推荐同时提供：

- 站点名称
- 站点地址
- 一句简介
- 是否允许 iframe 嵌入

这样可以同时保留“站内浏览”和“跳转访问”两种入口。
