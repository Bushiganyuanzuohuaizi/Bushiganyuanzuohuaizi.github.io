# 贡献说明

这份页面是站内版的贡献说明，对应仓库根目录的 `CONTRIBUTING.md`。

目前这个站点主要围绕两个方向维护：

- `牢邓的翘课日记` 本体内容
- 与 `EndeavourCHN` 等外部站点的联动页面

## 适合提交的内容

- 修正文案、排版和死链
- 补充或整理课程笔记
- 调整导航和栏目结构
- 优化首页或专题页体验
- 接入新的站外页面

## 最简单的更新方式

1. 把新文件放进 `docs/` 的对应目录。
2. 如果这是一个新页面，把它加入 `mkdocs.yml` 的 `nav`。
3. 如果这是一个新栏目，先补一个 `index.md`。
4. 预览一下页面，确认链接和排版没有跑掉。

## 外站集成约定

如果要在本站挂一个外部站点，最好同时准备：

- 站点名称
- 访问地址
- 简短介绍
- 是否允许 iframe 嵌入

这样即使浏览器或目标站点限制嵌入，页面里也还能保留一个可直接访问的入口。
